## Visual Studio 2019 Launch Event @Zaragoza by DotNetters
### Novedades Xamarin: Optimizaciones y librerías esenciales

Aquí están los materiales usados en la charla que realicé para DotNetters durante el [evento de lanzamiento del Visual Studio 2019](https://www.meetup.com/es-ES/dotnetters/events/259073780/).

- La presentación no tiene mucho mérito, es un recorte de [la de James Montemagno para el evento original de Microsoft](https://github.com/microsoft/VisualStudio2019Launch/blob/master/Technical/Mobile%20Development%20Simplified%20with%20Xamarin.pptx)

- Los proyectos [Xam17](https://github.com/Kash0321/VS19LE/tree/master/Xam17) y [Xam19](https://github.com/Kash0321/VS19LE/tree/master/Xam19) sirven para comprobar las optimizaciones en tiempo de compilación de VS2019 con respecto a VS2017.

- El proyecto [XamEssentials](https://github.com/Kash0321/VS19LE/tree/master/XamEssentials) tiene ejemplos de uso de las librerías Xamarin.Essentials

- El proyecto [XamShell](https://github.com/Kash0321/VS19LE/tree/master/XamShell) tiene un ejemplo muy simple del uso del nuevo Shell de Xamarin que está todavía en 
versión preliminar, básicamente la plantilla que viene por defecto, y que iba a usar para realizarle modificaciones y mostrar algunas novedades, aunque no dió tiempo a verlo.
